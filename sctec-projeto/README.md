# SCTEC - Entrega 1

Pacote da Entrega 1 do projeto **Sistema de Controle de Telescópio Espacial Compartilhado (SCTEC)**.

## Arquivos

- `MODELOS.md`: definição das entidades, relacionamentos e regras iniciais.
- `API.md`: especificação dos endpoints REST, payloads, respostas, códigos HTTP e HATEOAS.
- `LOGGING.md`: formato de logs de aplicação e logs de auditoria.
- `requirements.txt`: bibliotecas Python previstas para o serviço Flask e para o script de teste.
- `pdf/`: versões em PDF dos documentos Markdown.

## Observação

A Entrega 1 é uma entrega de especificação. O código funcional do serviço Flask, teste de estresse e demonstração da condição de corrida pertencem à Entrega 2.
